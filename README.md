# Art-Store
Demonstrando a implementação de enumeradores, structs e criando instâncias de produtos de uma loja em C#. Eu fui inspirado a criar esse projeto por uma amiga artista bastante talentosa, julguei que seria muito útil criar um sistema que pudesse armazenar as informações sobre as obras de um artista independente. O programa funciona armazenando os diferentes objetos de um mesmo tipo, no caso obras de arte, eu crio instâncias dessas obras de arte, tenho uma struct que armazena as propriedades de cada obra e dentro dela um método construtor, também criei um enumerador ao final para categorizar as obras de arte com um número para cada categoria, isso ajuda na organização e no processo de encontrar as obras. Na main é onde tudo acontece, eu crio as obras, passo seus respectivos parâmetros, e os imprimo na tela. Esse é apenas um esboço inicial do que eu pretendo fazer, farei um projeto completo futuramente para desenvolver todo o sistema, vai incluir o uso de ASP.NET, SQL Server e API.
